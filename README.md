# Kicks Kenya 👟

**Kenya's premier online sneaker destination** — a fully responsive multi-page web application built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

🌐 **Live site:** https://leonekamau.github.io/Kicks-Kenya
📁 **Repository:** https://github.com/leonekamau/Kicks-Kenya

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, featured products, categories, testimonials |
| Shop | `shop.html` | Filterable product grid by brand, category and price |
| Product Detail | `product.html` | Size selector, image gallery, reviews, add to cart |
| About | `about.html` | Brand story, team, values, stats |
| Contact | `contact.html` | Validated contact form, store info, M-Pesa details |
| Blog | `blog.html` | Sneaker culture articles and news |
| FAQ | `faq.html` | Accordion FAQ covering delivery, returns, sizing |
| Testimonials | `testimonials.html` | Customer reviews with rating breakdown |

---

## JavaScript Features

1. **Product Filter** — filter by brand, category and price range with live count
2. **Contact Form Validation** — validates name, email, Kenyan phone number and message
3. **Dark / Light Mode Toggle** — persists across all pages using localStorage
4. **Size Selector** — interactive size buttons on product detail page
5. **Add to Cart Toast** — confirmation message after selecting size
6. **Back to Top Button** — appears after scrolling 400px
7. **Scroll Animations** — fade-in on cards using IntersectionObserver

---

## Tech Stack

- **HTML5** — semantic markup, ARIA attributes, breadcrumb navigation
- **CSS3** — custom properties, flexbox, grid, media queries, dark mode
- **JavaScript ES6+** — localStorage, IntersectionObserver, arrow functions
- **Bootstrap 5.3** — grid system, navbar, accordion, tabs
- **Bootstrap Icons 1.11** — icon set throughout the site
